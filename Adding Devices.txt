Adding Devices to API:

1) Tap '+' in app device screen to add new device
2) Enter url of device/system json. This could be simplified using QR codes or having a central list of devices to be selected from a list.
3) App parses json with list of devices, actions, conditions to add
4) Select room, device name, and other params
5) App adds all of the devices (with accompanying actions and conditions) to API as shadow devices.
	Also searches for existing devices -> Prompts user if device type already exists:
	Add new device or connect to existing
6) App instructs user to connect to device wifi network
7) App confirms correct device is connected "Device found" (device hosts server in AP mode)
8) App sends device wifi ssid, pswd, device url, api key
9) Repeat for each device in array
